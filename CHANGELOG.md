# Changelog

## v2.15.0

- 修复移动端日期输入下方出现巨大空白。
- Flatpickr 移动端改为浮层打开，不再静态占位。
- 日期输入框只读并通过 touchstart / pointerdown 打开日期选择器，避免 iOS 键盘。
- 弹窗打开时隐藏右下角悬浮按钮。
- 加强横向滑动拦截，防止编辑弹窗右滑偏移。


## v2.15.0

- 修复移动端日期选择器弹出键盘。
- 日期选择器输入框改为只读，并在打开时自动 blur。
- 移动端 flatpickr 使用静态内嵌布局，减少遮挡和跳动。
- 修复移动端编辑弹窗横向右滑导致页面偏移的问题。
- 增加弹窗横向滑动拦截和页面横向滚动锁定。


## v2.15.0

- 修复移动端编辑弹窗滑动时弹来弹去的问题。

- 新增弹窗内部滚动边界控制。
- 新增 / 编辑、人员配置、Cloudflare 配置、链接预览弹窗均固定化。
