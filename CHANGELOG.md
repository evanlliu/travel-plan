# Changelog

## v2.37.0

- Plan Content 输入框自适应剩余高度。
- Red Note 输入框高度增加并自适应。
- 移动端编辑页按屏幕高度自动分配输入区域，减少空白。
- 保持默认无滚动，极小屏幕下才启用滚动兜底。


## v2.37.0

- 移动端编辑页改为每个输入项独占一行。
- Plan Content 自适应填充剩余高度，突出核心内容。
- 减少空白区域，让编辑页铺满整个屏幕。
- 默认隐藏滚动，仅在极小屏或键盘弹出时启用滚动兜底。
